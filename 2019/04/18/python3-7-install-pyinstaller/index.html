Python3.7安装pyinstaller

Author: we1x4n Date: April 18, 2019  23:53:53 Category: 瞎搞

0x01 前言

前几天换了mac，然后装了windows的虚拟机顺势就装了python的3.7版本

今天写了一个poc在编译的时候发现pyinstaller无论如何安装都会报错

google了一下发现似乎pyinstaller并没有支持python3.5以上的版本

0x02 安装

由于是新装的虚拟机而且我平时写代码并不再windows上，有的用户pychram安装的方法我都没有尝试

我就直接说我安装成功的方法

  1. 前往pyinstaller的GitHub项目下载pyinstaller
  2. 解压后cd到pyinstaller的文件夹然后执行python setup.py install
