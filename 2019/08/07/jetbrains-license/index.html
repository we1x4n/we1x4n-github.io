jetbrains-license

Author: we1x4n Date: August 7, 2019  21:10:21

0x01前言

自从pychram更新了2019之后，就不支持原来的激活方式了，找到了一个一个新的激活方式，写在博客上留个纪念

0x02激活

百度云链接提取码jd86

下载后将压缩包里面的jar文件存储于本机的任意位置

然后使用编辑器本身编辑vm options

在最下面添加-javaagent:/Users/weixun/jetbrains-agent.jarjar文件要写绝对路径

保存后激活，激活时可以选择使用我的code，或者是去这里获取你独立的code。生成自己的code的前提是你拥有自己的github账号，且github账号使用超过7天
