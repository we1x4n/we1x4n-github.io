利用cdn加速cod战区下载(适用github加速)

Author: we1x4n Date: March 18, 2020  18:50:33

最近准备玩cod战区，但是下载发现速度很慢，我发现了一个可以利用cdn加速下载方式。这个工具同理支持通过cdn加速github的访问。

下载地址：

cdn以及hosts修改工具

cdn重定向工具

使用方法：

现在打开UsbEAm Hosts Editor，在左下角选择战网Battle.net-暴雪战网，然后选择战网下载（国际服） - cdnworks


点击检测延迟选择第一个延迟最低的ip点击应用选中

然后打开战网下载cdn重定向，选择cdnworks，点击启动重定向然后启动战网就行了

最后总结一下：先测速，然后修改hosts，接着重定向，最后开车下载

使用方法：

在UsbEAm Hosts Editor，左下角找到github，同理可以加速github
